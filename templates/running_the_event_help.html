Running an event

Click on the card to be brought to the page it is referencing.

Setting Up The Arena

Event Settings Tab Size of Event and Alliances

Under the 'Event' tab, you can set the Event Name, Playoff Type (Double or Single eliminiation), number of Alliances, selection order, and if unpicked teams should be displayed. You can also set if you want to download team data from The Blue Alliance, optionally set an admin password, and backup and clear the database.

Game-Specific Tab Match Durations and RP

On the 'Game' tab, you can set the duration of Autonomous, Pause between Auto/Tele, Teleop, and the warning for the Endgame. You can also set the thresholds for game specific RP here.

Field Settings Tab Networking and PLC/IO

On the field settings tab, you can enable and configure advanced networking (AP, AP Channel, L3 Switch), configure the PLC address, enable alternate IO, and set the team sign display IDs if they are in use. You can also enable DS Lite mode to allow driver stations to keep the ability to control their own robot (E-Stop on space bar).

Publishing and Automation Tabs TBA and Nexus settings

On the 'Publishing' tab, you can set your TBA Event Code/Auth ID/TBA Secert to allow for publishing to TBA. On the 'Automation' tab, you can enable pulling lineups from Nexus, and can configure a Blackmagic HyperDeck to auto start/stop match video recording.


Setup Teams at Event, Match Schedule, and Testing

Team List Page Import Teams

Here, you can enter the teams at the event. Enter one team per line in the box. Effectively, you enter a team, and press enter. No comma or spacing required. Once completed, press 'Add Teams'. If you have TBA Download enabled, it will take a few minutes to populate the data.

Edit Team Info Edit Button

If you do not import data from TBA, or need to correct a team's information, you can select the Pencil icon on the right hand side of the team row. You can then enter the team name, nickname, school name, city, state, country, rookie year, robot name, recent accomplishments, and can enter if they've connected to the field.

Match Scheduling Page Schedule Parameters

On the match scheduling screen, you will create the practice (optional)/qualification schedule blocks. You start by entering the start and end time of the block, and then configure your cycle time. (Time from start of match to beginning of next). If you have another time frame (e.g. after lunch), click 'Add Block' and repeat the process. Once completed, click 'Generate/Save Blocks'. Once you have completed your schedule, click 'Save Schedule'.

Field Testing Page Making sure it works!

On the Field Testing screen, you can see if your PLC inputs are trigged or PLC coils are triggered. You can also test that the game sounds are playing correctly (e.g. make sure they are loud enough and playing in the right places!)


Match Play

Match Setup Pre-Match

Before a match can be started, all robots and driver stations must show successfully connected (green) on the Match Play screen. Once you are ready to start the match, you click the 'Start Match' button which will start moving robots. Make sure that all gates are closed and that all field personnel are clear of robots and moving objects.

Active Match During Match

Once the match has been started, it will run through 3 phases. Autonomous - Pause - Teleop. In Autonomous, robots are running pre-programmed routines. There is then a configured pause (usually 3 seconds). Robots then enter teleop (driver control), where robots are operator controlled. Once the end of the configured duration for teleop is reached, the match is ended and all robots are disabled. If something goes sideways, you can 'Abort' the match to stop the field at that time. This can be executed at any time during the match.

Post Match Post-Match

Once the match has been completed, you will need to 'Commit' the scores. This should only be completed once the head referee has given the OK. If additional time is needed to count scores, select the 'Count' button which will toggle the alliance station displays to indiciate that counting is underway. Once ready, you will need to click the 'Show Final Scores' button, which will change the audience display to show the scores. Make sure the announcer is ready!

Editing Scores Other Situations

In the event that scores need to be edited, you can select the "Edit Results" button which will give you the option to edit any score that has been entered. Once done editing, press 'Save' at the bottom to finalize the edit. You will then commit the scores the same way as a normal match. If you are outside of the match, you can go to Match Review, and select "Edit" on the respective match. Be sure to save!


Reporting

Team Information/Schedules Team List and Schedules

To get a team list, select the reports dropdown (category PDF reports) and then select 'Team List'. This will bring up a PDF report containing the list of configured teams. To get a practice or qualification schedule, select the reports dropdown (category PDF reports) and then select either Practice Schedule or Qualification Schedule.

Playoff reports Schedule, Alliances, and Bracket

To get a playoff schedule, select the report dropdown (category PDF reports) and then select 'Playoff Schedule'. This will bring up a PDF report of the playoff matches. To get a playoff bracket, select the report dropdown (category PDF reports) and then select 'Playoff Bracket'. This will bring up a PDF report of the playoff bracket. To get a alliance list, select the report dropdown (category PDF reports) and then select 'Playoff Alliances'. This will bring up a PDF report of the playoff alliances.

Alliance Station Preferences Playoff Station Selections

There are two ways to get sheets/coupons to hand out to alliance captains for selecting their preferred driver station during playoffs (or to sub out with a backup team). You can either select Help > DriverStation Preferences which will bring up a PDF of 4 driver station preference cards. You can also select reports > Playoff Alliance Coupons after alliance selection has been completed to get an alternate format of a similar report.

Backup Teams Playoff Substitutions

During playoffs, alliance captains have the option to sub out a robot for a backup should they feel the robot is no longer capable of performing at it's peak capability. Generally, the highest ranked remaining 4-8 teams are asked to stay as backups during playoffs. If a team needs to sub out with a backup, they are issued the next highest ranked available team as a backup. You can obtain a list of backups by selecting reports > Backup Teams.


Alliance Station Displays

Alliance Station Displays Pre-Match

Alliance Station Displays will show a team's number and a team's name/nickname. It will be either red or blue based on the team's respective alliance. If a team is disabled, it will show 'DISABLED' under their team number.

Alliance Station Displays During Match

Alliance station displays will change to show the red alliance score, blue alliance score, and the remaining match time. The middle display will show the time.

Alliance Station Displays Post-Match

Alliance station displays will show the team number until the head-ref signals for 'Count' or 'Field Reset'. At this point, the driver station may flash amber or be solid amber to indiciate an A or E stop trip.

Alliance Station Displays Continuous

Alliance station displays will also play game sounds.


Audience Displays

Audience Displays Pre-Match

Audience displays will show the 'Match Intro' display. This shows the teams that are on the field, their avatar (if available), the match type and number, and the event name.

Audience Displays During Match

Audience displays will show the 'Match Play' display. This shows the teams that are on the field, their avatar (if available), the match type and number, event name, each alliance's scores, and the game time remaining.

Audience Displays Post-Match

The audience display will show the 'Blank' display once the match completes. Once the scores have been committed, and 'Show final score' has been selected, the audience display will then change to the 'Final Score' display. This will show the outcome of the match, in addition to the score breakdown and rankings of teams that are in the match.

Audience Displays Other Functions

Audience Displays will always play game sounds. You can also choose to show the playoff bracket, alliance selection display, game logo, or sponsor reel (if configured).


Alliance Station E-Stop/A-Stop

E-Stop/A-Stop Pre-Match

From the A-Stop/E-Stop panel, teams can be A-Stopped or E-Stopped. If a team is A-Stopped prior to a match, they will be able to resume at teleop/driver control. A team cannot be E-Stopped prior to a match, or the match will not be able to be started.

E-Stop/A-Stop During Match

If a team A-Stops during autonomous, the robot is disabled for the remainder of the Autonomous period. The A-Stop cannot be used outside of autonomous. If a team E-Stops at any time during the match, the robot is disabled for the remainder of the match and cannot be re-enabled.

E-Stop/A-Stop Post-Match

Once the match has ended, all E-Stops and A-Stops will need to be physically reset. This is done by unlocking/releasing the button that has been pressed. This can be identified by an amber flashing driver station display (A-Stop), or a solid amber driver station display (E-Stop).

E-Stop/A-Stop Other Functions

Optionally, all teams can be A-Stopped or E-Stopped from an FMS panel (Panels > E-Stops/A-Stops > Red or Blue). This can be used if an event does not have the physical hardware available to provide A-Stops or E-Stops in a driver station.
